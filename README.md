# Tableau-Dashboards
Welcome to the Tableau Dashboard Repository by Hiran Joseph.

## Introduction
This repository hosts a variety of Tableau projects and dashboards,

## Project Structure
Each project in this repository is structured the same way, some will have additional files.

- **[Project Name]**: Descriptive name for the Tableau project.
  - <project_name>`.twb`: Tableau workbook file containing the dashboard.
  - data file formated as `.csv` or `.xlsx`: The dataset used for analysis.
  - <presentation_name>`.pptx`: A PowerPoint presentation summarizing the project's findings.
  - `README.md`: Detailed information about the project, including its objectives and data sources.

## Getting Started
To explore and analyze a specific Tableau project, follow these steps:

1. Clone this repository to your local machine:

     git clone https://github.com/manolincabral/Tableau-Dashboards
  
2. Navigate to the project folder you are interested in.

3. Open the `project_name.twb` file using Tableau Desktop to interact with the dashboard and explore the analysis.

4. Examine the data file to understand the dataset used in the project.

5. Review the presentation file to get an overview of the project's key findings and insights.

6. Explore the `README.md` for in-depth project details and context.
   
## Projects in this Repository
- **Customer Analysis Dashboard**: Analyze customer behavior and purchasing patterns.
- **Real Estate Market Dashboard**: Informational dashboard about the real estate market in 10 cities for the year 2023.
- **HR Dashboard**: Comprehensive dashboard to analyze human resources data, providing both summary views for high-level insights and detailed page for employee records.
